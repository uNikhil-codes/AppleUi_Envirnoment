Make your VScode look premiuim !!
Follow the steps below to apply it.
---
### 1️⃣ Open VS Code

---
### 2️⃣ Open Settings (JSON)
Use **any one** method:
* Press
  **`Ctrl + Shift + P`**
* Search for
  **`Preferences: Open Settings (JSON)`**
* Press **Enter**
This will open your current `settings.json`.
---
### 3️⃣ Copy the Settings
* Open the `settings.json` file from this GitHub repository
* Copy **all** its contents
---
### 4️⃣ Paste and Save
* Paste the copied code into your VS Code `settings.json`
* Press **`Ctrl + S`** to save
VS Code will apply the settings instantly.
---
## Recommended (Optional)
For best results, install these:
* **Font**: JetBrains Mono or Fira Code
* **Extensions**:

  * Codium / Codiumate
  * Material Icon Theme
  * One Dark Pro (or any dark theme)

Restart VS Code after installing fonts.

---

## Notes

* A large `settings.json` file **does not affect performance**
* If VS Code shows errors, recheck commas and quotes
* You can remove any setting you don’t like
---

## Reset (If Needed)

To undo everything:
* Clear the contents of `settings.json`
* Save the file
VS Code will return to default behavior.
